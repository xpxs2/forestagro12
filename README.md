
# ForestAgro-12

This is a [Next.js](https://nextjs.org/) project for ForestAgro-12, a company focused on sustainable solutions that combine forestry and agriculture.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Firebase Integration

This project is integrated with Firebase. To connect to your Firebase project, you will need to add your Firebase configuration to `src/firebase.ts`.

Replace the placeholder values in the `firebaseConfig` object with your actual Firebase project credentials.

## Project Structure

- `src/app/page.tsx`: The main page of the application.
- `src/app/icons.tsx`: A collection of SVG icons used in the application.
- `src/firebase.ts`: The Firebase configuration file.
